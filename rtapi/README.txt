= ruby-tumblr

 http://ruby-tumblr.rubyforge.org/

== DESCRIPTION:

ruby-tumblr is a library for tumblr API.

== FEATURES/PROBLEMS:

Unknown

== SYNOPSIS:

<3 ruby + <3 tumblr

== REQUIREMENTS:

To run Rake:

hoe
newgem
tzinfo
pit 
mechanize (libxml2-dev and libxslt-dev required)
rspec

== INSTALL:

sudo gem install hoe newgem tzinfo pit rspec mechanize


% pit set tumblr.com
